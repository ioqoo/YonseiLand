Syllabus_Korean
##
본 강의는 산업공학 분야의 정보 분석을 위한 프로그래밍 능력 함양을 목표로 한다. 정보 처리 및 데이터 분석 과정에서 의사 결정의 도구로 python을 활용할 수 있도록, python의 기본적인 사용 방법을 이해한 후, 기술 통계 (그래프), 추론 통계 (검정), 데이터 분석을 위한 이론과 함께 python의 주요 라이브러리의 활용 방식을 학습한다.

Syllabus_English
##
This course aims to develop the programming ability in industrial engineering field. In order to utilize python programming as a tool for decision making in information processing and data analytics, it studies programming basics, statistics (graph and test), data analysis. (It includes the usage of python packages.)

Attendance 10% / Programming assignment (or Quiz) 20% /  Midterm exam 30% / Final team project 40%
